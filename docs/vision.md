# STAGELINK: Project Vision & Master Plan

## 1. Core Concept
StageLink is the "Discovery Hub for Theater." A social utility platform where audiences discover shows, buy tickets, and stay connected with the local theater scene.

## 2. The "Social Portal" (Landing Experience)
* **Vibe:** "The Lobby." Dark mode, elegant, buzzing with activity.
* **Landing Page:**
    * **Hero Section:** "Your Stage. Your Community." (Not just "Buy Tickets").
    * **Social Proof:** "Join theater lovers across the Philippines."
    * **Live Activity Feed:** Scrolling updates (e.g., new reviews, newly listed shows, trending productions).
* **Login/Signup:**
    * **Style:** Split-screen design (Left: High-quality theater art / Right: Clean form).
    * **Call to Action:** "Claim your Profile" instead of just "Register."

## 3. Product Direction (No Gamification)
StageLink will **not** implement XP, levels, rank ladders, or badge progression.

Instead, user value is centered on:
* **Personalized Discovery:** Better recommendations by city, favorites, and behavior.
* **Audience Identity:** Display names, profile bios, and credible participation.
* **Community Proof:** Reviews/ratings and visible recent activity.
* **Retention Loops:** Saved-show reminders, city/group updates, and schedule alerts.
* **Producer Outcomes:** Better analytics insights that improve show promotion decisions.

## 4. Feature Priorities (Updated)

### Phase A: Audience Identity & Social Proof (FIRST PRIORITY)
* Add audience display name and profile bio support.
* Improve review/rating visibility in high-traffic pages.
* Highlight social proof modules (popular this week, recently reviewed, recently listed).

### Phase B: Discovery & Retention
* Build recommendation modules based on city + favorites + behavior.
* Add follow/reminder notifications for saved shows and followed groups.
* Add reliability improvements for loading/empty/error states.

### Phase C: Producer Growth Tools
* Expand analytics beyond basic views/clicks to practical producer insights.
* Add clearer conversion signals and trend summaries.
* Strengthen producer-facing calls to action tied to measurable outcomes.
